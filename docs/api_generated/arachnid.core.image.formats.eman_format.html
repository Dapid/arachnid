eman_format¶

Use EMAN2/Sparx to read or write to a file

The following formats are supported for reading:

MRC IMAGIC
SPIDER HDF5
PIF ICOS
VTK PGM
Amira Video-4-Linux
Gatan DM2 Gatan DM3
TIFF Scans-a-lot
LST PNG

The image writing interface selects the image format based on the file extension. The following table lists each supported extension with its corresponding file format.

Extension Format   Extension Format
rec, mrc, MRC, ALI, tnf, TNF, ccp4, map MRC   img, IMG hed, HED imagic IMAGIC IMAGIC
spi, SPI, spider, SPIDER Spider Stack   spidersingle SPIDERSINGLE singlespider SINGLESPIDER Spider Image
jpg, JPG, jpeg, JPEG JPEG Image Format   png, PNG Portable Network Graphics
vtk, VTK Visualization Toolkit   xplor, XPLOR X-Plor File Format
h5, H5, hdf, HDF Hierarchical Data Format   map, MAP icos, ICOS ICOS Image Format
am, AM, amira, AMIRA AMIRA Image Format   pif, PIF Portable Image Format For EM data
lst, LST lsx, LSX List of Image Files   pgm, PGM Portable Gray Map
