mask_volume¶

Mask a volume

This pySPIDER batch file (spi-mask) generates a masked volume. It supports:

  1. Smoothed spherical masks
  2. Adaptive tight masks
  3. Existing masks in a file

Tips¶

  1. A good value for the density threshold can be found manually in Chimera
  2. The size of a spherical mask is the pixel-diameter from the SPIDER params file
  3. The mask-edge-width also increases the size of the mask by the given number of pixels
  4. Setting –volume-mask to N in this script is an error

Examples¶

# Adaptively tight mask a volume

$ spi-mask vol01.spi -o masked_vol01.spi --volume-mask A

# Mask a volume with a cosine smoothed spherical mask

$ spi-mask vol01.spi -o masked_vol01.spi --volume-mask C --mask-edge-width 10

# Mask a volume with a mask in a file

$ spi-mask vol01.spi -o masked_vol01.spi --volume-mask mask_file.spi

Critical Options¶

-i <FILENAME1,FILENAME2>, --input-files <FILENAME1,FILENAME2>, FILENAME1 FILENAME2¶

List of input filenames containing volumes. If you use the parameters -i or –inputfiles they must be comma separated (no spaces). If you do not use a flag, then separate by spaces. For a very large number of files (>5000) use -i “filename*”

-o <FILENAME>, --output <FILENAME>¶

Output filename for masked volume with correct number of digits (e.g. masked_0000.spi)

-mask <('A', 'C', 'G' or FILENAME)>¶

Set the type of mask: C for cosine and G for Gaussian and N for no mask and A for adaptive tight mask or a filepath for external mask

-p <FILENAME>, --param-file <FILENAME>¶

Path to SPIDER params file

--bin-factor <FLOAT>¶

Number of times to decimate params file

Spherical Mask Options¶

--mask-edge-width <INT>¶

Set edge with of the mask (for Gaussian this is the half-width)

Adaptive Tight Mask Options¶

--threshold <STR or FLOAT>¶

Threshold for density or ‘A’ for auto threshold

--ndilate <INT>¶

Number of times to dilate the mask

--gk-size <INT>¶

Size of the real space Gaussian kernel (must be odd!)

--gk-sigma <FLOAT>¶

Width of the real space Gaussian kernel

--pre_filter <FLOAT>¶

Resolution to pre-filter the volume before creating a tight mask (if 0, skip)

Other Options¶

This is not a complete list of options available to this script, for additional options see:

  1. Options shared by all scripts ...
  2. Options shared by |spi| scripts...
  3. Options shared by file processor scripts...
  4. Options shared by SPIDER params scripts...
